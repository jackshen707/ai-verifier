🔍 AI 核实器(AI Verifier)
一款 Chrome 浏览器插件:在任意网页选中文字或右键图片,即可调用 DeepSeek / 豆包 网页版 AI 进行事实核实、翻译、总结、解释——无需 API Key,使用你浏览器中已登录的 AI 网站会话。
✨ 功能特性
右键菜单功能说明🔍 用 AI 核实选中内容核查新闻、传言的真实性,输出可信度评分(0-100)、判定结论、分析理由和核实建议🖼️ 用 AI 核实这张图片自动抓取并上传图片,识别图中文字与内容,判断是否造假或 AI 生成🌐 用 AI 翻译成简体中文将选中的外语文本翻译成自然流畅的简体中文📝 用 AI 总结这段话一句话概括 + 分点要点💡 用 AI 解释一下用大白话解释术语、代码、合同条款等,附带例子和比方
其他亮点:

🆓 无需 API Key——直接复用你已登录的 DeepSeek / 豆包网页会话
🪟 结果显示在当前页面右下角的悬浮面板,无需切换标签页,可一键复制
🖼️ 图片防盗链对策——自动携带 Referer + 页面 Canvas 提取双重方案,微博等防盗链图床也能核实
🔁 发送自动重试、面板脚本自动注入,稳定性经过多轮打磨
🔧 网站改版时,只需修改 ai-runner.js 顶部集中的「选择器配置区」

📦 安装方法

点击本页绿色 Code 按钮 → Download ZIP,下载后解压
打开 Chrome,地址栏输入 chrome://extensions 回车
打开右上角「开发者模式」开关
点「加载已解压的扩展程序」→ 选择解压出的文件夹
点击插件图标进入设置页 → 选择 DeepSeek 或豆包
首次使用前,先在浏览器中登录一次对应的 AI 网站
刷新目标网页,右键即可使用!

🚀 使用方法

在任意网页选中一段文字,右键 → 选择「AI 核实器」下的功能
或在图片上右键 → 「🔍 用 AI 核实这张图片」
插件会在后台标签页自动操作 AI 网站,结果显示在当前页面右下角的悬浮面板


💡 核实图片建议在设置中选择豆包(识图能力较强);纯文字任务两者均可。

🛠️ 工作原理
选中内容/右键图片
      │
      ▼
后台脚本构建提示词(图片则先抓取数据)
      │
      ▼
自动打开/复用 DeepSeek 或豆包标签页(后台,不打扰浏览)
      │
      ▼
自动填入问题 → 模拟上传图片 → 自动发送
      │
      ▼
监测 AI 回答(内容稳定 4.5 秒视为完成)
      │
      ▼
结果回传 → 原页面悬浮面板展示
📁 文件结构
ai-verifier/
├── manifest.json    # 插件配置清单(Manifest V3)
├── background.js    # 后台调度:右键菜单、提示词、图片抓取、消息路由
├── ai-runner.js     # AI 网站自动化:找输入框、发送、抓取回答
├── panel.js         # 结果悬浮面板(全站注入)+ 备用抓图
├── options.html     # 设置页面
└── options.js       # 设置页脚本
❓ 常见问题
Q:面板提示"未找到聊天输入框"?
A:多数是 AI 网站登录过期了,点面板上的「查看 AI 页面」登录后重试。
Q:发送失败?
A:常见原因是 AI 还在回答上一条消息、或图片仍在上传,稍等几秒重试即可。
Q:DeepSeek / 豆包改版后插件失效?
A:打开 ai-runner.js,顶部「选择器配置区」集中了所有页面元素选择器,按注释更新即可,欢迎提 Issue 反馈。
